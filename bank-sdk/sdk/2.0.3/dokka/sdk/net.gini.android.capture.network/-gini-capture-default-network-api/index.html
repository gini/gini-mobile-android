GiniCaptureDefaultNetworkApi

public final class GiniCaptureDefaultNetworkApi implements GiniCaptureNetworkApi

Default implementation of network calls which can be performed manually from outside the Gini Capture SDK (e.g. for sending feedback).

To create an instance use the GiniCaptureDefaultNetworkApi.Builder returned by the builder method.

In order to easily access this implementation pass an instance of it to GiniCapture.Builder.setGiniCaptureNetworkApi when creating a GiniCapture instance. You can then get the instance in your app with GiniCapture.getGiniCaptureNetworkApi.
