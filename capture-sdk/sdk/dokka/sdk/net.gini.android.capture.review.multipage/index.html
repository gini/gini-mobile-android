Package net.gini.android.capture.review.multipage

Contains the Activity and Fragment used for the Multi-Page Review Screen.

Screen API

The net.gini.android.capture.review.multipage.MultiPageReviewActivity displays images of a document and allows the user to reorder, rotate, zoom and delete them.

Component API

To use the Component API you have to include the net.gini.android.capture.review.multipage.MultiPageReviewFragment in an Activity in your app (a dedicated Activity is recommended). To receive events from the Fragments your Activity must implement the net.gini.android.capture.review.multipage.MultiPageReviewFragmentListener interface.
